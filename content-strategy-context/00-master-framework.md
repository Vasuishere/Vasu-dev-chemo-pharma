# Master Framework — Writing Guidelines, Competitor Analysis & Quality Checks

## 1. Writing Guidelines (Apply to All Countries)


### Voice & Tone
- **Active voice** throughout; avoid passive constructions
- **Professional B2B tone** — write for procurement managers, field engineers, plant operators
- **Neutral and factual** — no marketing hyperbole, no unverified claims
- **Credible** — cite industry standards (API, NACE, ISO) where relevant

### Formatting Rules
- Short paragraphs (3–4 sentences max)
- Bullet points for lists of features, specs, applications
- Use subheadings (H2/H3) every 150–200 words
- Bold key terms on first use
- Tables for specifications and comparisons

### Keyword Usage
- Primary keyword in: title, H1, first 100 words, meta description
- Long-tail keywords distributed across H2/H3 headings
- Keyword density ≤ 1.5% — prioritize semantic coverage over repetition
- Include local-language equivalents in parentheses where applicable

### Localization Notes
- Default language: **English** for all markets
- For Arabic-speaking markets (UAE, Saudi, Oman, Qatar, Kuwait, Iraq, Libya, Egypt, Jordan): provide Arabic keyword equivalents, note RTL layout considerations
- For non-English markets: include a "Native Language Translation Plan" section with key terms translated
- Do NOT machine-translate full articles — use professional translators or native speakers

## 2. Competitor Analysis Framework (Apply Per Country)

### Step 1: Identify 3–5 Competitors
For each target country, identify the top 3–5 pages ranking for:
- "[Country] H2S scavenger supplier"
- "MEA triazine [Country]"
- "H2S scavenger manufacturer [Country/Region]"

### Step 2: Extract & Analyze

| Analysis Point | What to Record |
|---------------|---------------|
| **URL & Domain Authority** | DA/DR score via Ahrefs/Moz |
| **Page Title & Meta** | Exact title tag and meta description |
| **Content Length** | Word count of main body content |
| **Heading Structure** | Full H1–H3 outline |
| **Content Sections** | What topics are covered (specs, applications, safety, FAQs) |
| **Tone & Language** | Technical depth, use of local language |
| **Schema Markup** | Product, FAQ, Organization schema present? |
| **Backlink Profile** | Number and quality of referring domains |
| **Visual Assets** | Images, diagrams, videos, PDFs |
| **CTAs** | Quote request, download, contact forms |

### Step 3: Differentiation Strategy
For each competitor, document:
1. **Gap:** What they cover poorly or miss entirely
2. **Strength:** What they do well that we must match
3. **Opportunity:** Where we can outperform (deeper specs, local compliance info, dosing data, comparison tables)

## 3. Content Quality Checks

### Grammar & Punctuation
- [ ] No spelling errors (use Grammarly or equivalent)
- [ ] Consistent spelling (American English default; British English for UK-oriented markets)
- [ ] Proper chemical nomenclature (IUPAC-compliant)
- [ ] Numbers: use numerals for measurements and concentrations (78%, not seventy-eight percent)

### Readability Targets

| Market Language | Metric | Target |
|----------------|--------|--------|
| English | Flesch-Kincaid Grade Level | 9–12 (technical but accessible) |
| English | Flesch Reading Ease | 45–55 (suitable for industry professionals) |
| Arabic | — | Clear, modern standard Arabic; avoid overly literary style |
| Portuguese (BR) | Flesch-Kincaid adapted | Equivalent professional grade |
| Vietnamese | — | Clear, concise Vietnamese; confirm with native reviewer |
| Thai | — | Professional technical Thai; native review required |
| Turkish | — | Modern business Turkish; native review required |
| Russian | — | Technical Russian standard; native review required |
| French (Niger) | Flesch adapted | Professional scientific French |

### Credibility & Source Plan
- Cite industry standards: API RP 55, NACE MR0175, OSHA 29 CFR 1910.1000
- Reference SDS data (do NOT reproduce full SDS — link to downloadable PDF)
- Use "per industry data" or "based on field studies" for efficiency claims
- Never fabricate statistics or performance data
- Link to Vasudev Chemo Pharma product page as canonical source

### SEO Technical Checks
- [ ] Title tag ≤ 60 characters
- [ ] Meta description 145–155 characters
- [ ] Single H1 per page containing primary keyword
- [ ] All images have descriptive alt text with keywords
- [ ] Internal links to related products and resources
- [ ] Schema markup (Product, FAQPage, BreadcrumbList)
- [ ] Canonical URL set correctly

Every country brief follows this structure:

```
# [Country] — MEA Triazine 78% Content Brief

## 1. Topic & Keyword Map
- Primary keyword(s)
- 5–7 long-tail keywords
- Topical mapping (applications, regulations, logistics, safety)

## 2. Content Outline
### Headline Options (3 SEO-friendly options)
### Introduction (hook + intent, 80–100 words)
### Body Sections (H2/H3 structure)
  - Product overview
  - Applications specific to market
  - Benefits and technical data
  - Compliance/regulatory notes
  - Logistics and supply information
### FAQ Section (5–6 questions)

## 3. Competitor Analysis Notes
- Key competitors in this market
- Differentiation opportunities

## 4. Sample Content (2–3 paragraphs)

## 5. Localization Notes
- Native language keyword equivalents
- Translation priority and plan
```
